How I built this website

I built this website to create a personal space in the web and post parts of my albums, to give them more glory and credit. Also I wanted to save them from the conversion-alghoritms of social media and put them in a space I can call my own.

This is a static site built in less than a month during April 2020 with the open source framework Hugo, when the entire world was quarantined at home, so I took this time to focus on personal projects.
The main goal of this space is to publish my previous and future projects just like they are, with the purpose to create a sort of personal museum to observe the evolutionary process over time.

It uses a minimalistic theme I found on Github called LoveIt.
I tried to find a compromise between photo quality and loading speed of the website.
Hope you like it.
